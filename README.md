In this repository, I'll be solving self-assigned problems on neural network theory to practice writing low-level code and understanding the underlying mathematics. This repository was created to review and deepen my knowledge of neural networks, which I acquired while taking courses at MIPT. While that course provided a solid theoretical foundation, I created this repository to bridge the gap between theory and low-level implementation, as well as to master the mechanics of neural networks from scratch.

Currently, the repository contains the following tasks:
- Writing a neural network from scratch without using any libraries
- Writing a neural network using only the NumPy library, as well as writing a linear regression in NumPy and comparing them
